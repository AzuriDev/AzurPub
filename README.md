# AzurPub
HEY ! AVANT D'UTILISER CE SCRIPT, VEUILLEZ BIEN SUIVRE LES ETAPES CI DESSOUS ! :
- Télécharger les modules fs et discord.js via npm.
- Présisez votre token de bot dans ../config/config.js/.
- Présisez l'id du serveur ou les commandes seront exécutés avec le prefix dans ./config/prefixes.json/.
- Entrez votre message de publicité dans ./index.js/ (ligne : 61) lorsque quelqu'un rejoins un serveur ou le bot se retrouve, le bot envoira votre publicité a l'arrivant !
- Enfin, run 'node index.js' dans votre terminal!
Bonne pub!
